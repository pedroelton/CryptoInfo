## Technologies Used

- ReactJs
- Coinstats API

To test the project all you need to do is to clone the repository:

```sh
git clone https://github.com/pedroelton/CryptoInfo.git
```

Enter in the file:

```sh
cd crypto-price
```

Install all the dependencies:

```sh
npm install
```

or

```sh
yarn
```

Start the project:

```sh
npm start
```
